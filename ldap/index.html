LDAP Filter 產生器 (PVE 專用)

將路徑清單轉換為 User Filter (登入權限) 與 Group Filter (群組同步)。

  • 請注意 Windows AD 裡 DC、OU 跟 CN 的設定，成員必須在 CN 中：參考圖。
  • 如果不確定群組路徑清單，請使用 ldapsearch 指令搜尋：ldapsearch 指令產生機。

1 群組路徑清單

請貼上完整路徑，系統將視最後一節為群組(CN)。

格式：網域/OU/群組名稱 0 筆

2 特定帳號 (Optional)

1. User Filter (使用者過濾)

邏輯: memberOf=群組DN

2. Group Filter (群組同步)

邏輯: distinguishedName=群組DN
用於同步群組物件
